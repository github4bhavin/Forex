Forex.pm

Perl Module that give you Daily and Historic Foreign Exchange Rates for all currencies from OpenExchangeRates.org. You would need and account with them but its free for basic usage.


    use Forex;
    
    my $forex = new Forex( 'APP_ID' => $app_id, 'BASE' => 'USD');
    
    #__fetch and initialize daily rates
    #__from $from_date to $to_date in yyyy-mm-dd
    $forex->get_rates_from ( $from_date, $to_date );
    
    #___ fetches rates for $date (yyyy-mm-dd)
    $forex->get_rates( $date );
    
    #___ fetches reates for today
    $forex->get_rates();

Methods

Constructor

my $oxe = new Forex::OXR( OXR_HOME => 'https://openexchangerates.org',
              API_HOOK => 'api',
              APP_ID   => 'temp-e091fc14b3884a516d6cc2c299a',
              BASE     => 'AUD');

API_HOOK is the path used by OpenExchangeRates, if you have a paid service you might have a different path provided by OpenExchangeRates along with your private APP_ID.

get_rate_of

$rate = $oxe->get_rate_of('AUD','2012-09-10');

returns rate of $currency on $date with BASE default to 'USD'.This method actually looks up the cached rates from array $obj->{CURRENCIES}, which is filled by calling get_rates or get_rates_from() methods.'

get_rates_from

$oxe->get_rates_from('2012-07-10','2012-09-10');

Downloads and fills array $obj->{CURRENCIES}, from $from_date to $to_date . Dates are expected in yyyy-mm-dd format.

get_rates

$oxe->get_rates('2012-07-10');

Downloads and fills array $obj->{CURRENCIES} for $day specified. Dates are expected in yyyy-mm-dd format.

get_currency_symbols

$oxe->get_currency_symbols();

Downloads and initializes all currency symbols from OpenExchangeRates.org. Method is useful to findout supported currencies.

base_currency

$oxe->base_currency('AUD');
$base_currency = $oxe->base_currency();

sets or return current value of BASE_CURRENCY.

oxr_home

$oxe->oxr_home('https://OpenExchangeRates.org');
$oxr_home = $oxe->oxr_home();

sets or return current value of OpenExchangeRates's Home.You could youse this value to change the default http to https connection if you are a paying coustomer to OpenExchangeRates.org

app_id

$oxe->app_id('e091fc14b3884a516d6cc2c299a');
$app_id = $oxe->app_id();

sets or return current value of APP_ID.

get_currencies

$oxe->get_currencies();

returns all currencies from currency's hash.

last_error

$oxe->last_error();

returns last error object as returned by OpenExchangeRates.

last_error_message

$oxe->last_error_message();

returns last error message generated by Forex.pm module.
